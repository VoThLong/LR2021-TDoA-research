## What's In The Box

- 2x LR2021WIO1XDCS Wio-LR2021 Module for 868/915 MHz & 2.4GHz Bands - PCB Nr.: E769V01A
    
- 2x Seeed Studio XIAO nRF54L15 Module (MCU by Nordic)
    
- 2x LoRa Plus™ Expansion Board V2a Interface - PCB Nr.: E758V03A
    
- 2x LF 868 MHz Antenna (Europe – grey rings) – Linx Tech P/N: ANT-868-CW-HWR-SMA
    
- _or_ LF 915 MHz Antenna (US – yellow rings) – Linx Tech P/N: ANT-916-CW-HWR-SMA
    
- 2x HF 2.4 GHz Antenna – Pulse Electronics P/N: W1010 (2.4GHz – shorter antenna)
    
- 2x U.FL to U.FL RF coaxial cable 50mm length – for ANT_HF RF path (2.4 GHz)
    
- 2x U.FL to U.FL RF coaxial cable 100mm length – for ANT_LF RF path (868 or 915 MHz)
    
- 2x USB Cable Type-A Male to Type-C Male – 1m length or equivalent
    
- Read Me Note (this document)
